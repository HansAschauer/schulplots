# Installation

`schulplots` benötigt eine Python-Laufzeitumgebung, in der Version 3.9 oder höher. Diese findet man auf der offiziellen [Homepage](https://www.python.org/downloads/) des Python-Projekts.

Unter Linux ist oft Python schon vorinstalliert, oder lässt sich mit dem üblichen Paket-Manager installieren (`apt-get`, `zypper`, `yum`, `discover` -- je nach Distribution).

Daneben benötigt die Installation den Python-Paketmanager `pip`. Sofern dieser nicht ohnehin schon mitinstalliert wurde, kann man ihn, wie [hier](https://pip.pypa.io/en/stable/installation/) beschrieben, installieren.

Jetzt kann man `schulplots` installieren:

 `pip install schulplots`
