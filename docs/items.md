# Items, die zu einem Achsensystem hinzugefügt werden können

Die folgende Tabelle zeigt eine Aufstellung aller Item-Typen, die zu einem Achsensystem hinzugefügt werden können. 

| type    | Beschreibung | 
| ----    | ------------ |
| `Graph` | Der Graph einer Funktion |
| `Area`  | Fläche zwischen zwei Funktionen |
| `VLine` | vertikale Linie |
| `HLine` | horizontale Linie |
| `VSpan` | farblich hervorgehobener Bereich zwischen zwei x-Werten |
| `HSpan` | farblich hervorgehobener Bereich zwischen zwei y-Werten |
| `Arrow` | Pfeil | 
| `Text`  | beliebiger Text |
